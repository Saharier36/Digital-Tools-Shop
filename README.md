## ✨ Features

### 🛒 Add to Cart & Manage Orders
Users can add any product to the cart with a single click. The cart updates instantly and shows the total number of items in the navbar in real time. Users can also remove individual items or clear the entire cart at once.

### 📦 Dynamic Product Listing
Products are loaded dynamically from a JSON file, making it easy to update the catalog without changing any component code. Each product card displays the name, price, category, and a short description.

### 🔔 Toast Notifications
Every cart action — adding or removing a product — triggers a clean toast notification so users always know their action was successful. This improves the overall user experience and makes the app feel polished and responsive.
